# Draftboi
Discord bot that simulates a Paladins draft. Written in python with discord.py.

Not updated anymore due to the competitive scene being killed off and not having a need for it anymore. Uploading to github in case someone can get use out of it.

Useful info if you're interested in working on this sort of stuff:
https://discordpy.readthedocs.io/en/latest/
https://www.gngrninja.com/code/2017/3/24/python-create-discord-bot-on-raspberry-pi
